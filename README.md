# StreamZone

Projeto académico desenvolvido no âmbito da disciplina de Tecnologias Internet.

A StreamZone é um sítio Web estático que simula uma plataforma de streaming de música e vídeo, usando HTML5, CSS3, JavaScript, XML e XSD.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript
- XML
- XSD

## Estrutura

- `index.html` — página inicial
- `catalogo.html` — catálogo carregado a partir de XML
- `planos.html` — planos fictícios
- `multimedia.html` — informação sobre multimédia e streaming
- `contactos.html` — formulário de contacto

## Execução local

Abrir o projeto com Live Server no Visual Studio Code e aceder a `index.html`.

## Publicação

O site será publicado no Netlify.

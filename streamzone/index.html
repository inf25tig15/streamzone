Descobre música e vídeo num só lugar

A StreamZone é uma plataforma fictícia criada para demonstrar a utilização de tecnologias Web na construção de um sítio estático dedicado ao streaming multimédia.

O projeto utiliza HTML5, CSS3, JavaScript, XML e XSD.

Funcionalidades do projeto

Catálogo XML

O catálogo da plataforma é guardado num ficheiro XML e carregado para uma tabela HTML através de JavaScript.

Interface responsiva

A interface foi preparada com CSS3 para se adaptar a computadores, tablets e telemóveis.

Objetivo académico

Este site não representa uma plataforma real. É uma simulação estática desenvolvida para fins académicos, sem servidor, autenticação ou base de dados.

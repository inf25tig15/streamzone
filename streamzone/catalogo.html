Catálogo multimédia

Nesta página são apresentados conteúdos de música e vídeo carregados a partir de um ficheiro XML através de JavaScript.

O ficheiro XML funciona como uma pequena fonte de dados local. O JavaScript lê esse ficheiro, interpreta os elementos e cria automaticamente as linhas da tabela.
